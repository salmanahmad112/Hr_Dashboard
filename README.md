# Hr_Dashboard
HR data preprocessing and dashboard project using Python, Pandas, and Plotly to clean, transform, and visualize employee dataset insights.
This project focuses on preprocessing an HR dataset and creating a simple interactive dashboard for data analysis. The notebook includes data loading, basic inspection, duplicate checking and removal, missing value checking, categorical encoding, and visual exploration using Plotly charts. It helps in understanding employee-related patterns such as salary, satisfaction level, projects, and monthly working hours.
